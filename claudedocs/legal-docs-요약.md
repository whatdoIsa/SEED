## 1. 개인정보처리방침 (Privacy Policy)

SEED의 강점은 "거의 아무것도 수집하지 않는다"는 것 — 문서도 그 사실을 그대로 쓰면 됩니다.

### 필수 항목

**① 수집하는 정보: 사실상 없음**
- 회원가입 없음 — 이름·이메일·전화번호 등 계정 정보를 수집하지 않음
- 매매 기록·레슨 진행·시즌 데이터는 **사용자 기기와 사용자 본인의 iCloud(개인 저장소)에만** 저장됨. 개발자는 접근 불가
- 사용 통계(KPI 이벤트)는 기기 내 로컬 파일에만 기록되며 외부 전송 없음
- 광고 없음, 광고 식별자(IDFA) 미사용, 제3자 추적 SDK 없음

**② 유일한 외부 전송: AI 튜터 질문** ← 가장 중요, 빠지면 안 됨
- 사용자가 AI 튜터에 질문을 입력하면, **질문 텍스트가 서버(Cloudflare Workers)를 거쳐 AI 제공사(Anthropic)로 전송**되어 답변을 생성함
- 전송되는 것: 질문 텍스트, 기기별 익명 식별자(일일 사용량 제한용)
- 전송되지 않는 것: 이름·연락처·매매 기록·위치 등 개인 식별 정보
- 질문은 답변 생성 목적으로만 처리되며, 개발자가 별도 보관·판매하지 않음
- (Anthropic의 API 데이터 정책상 학습에 사용되지 않음을 명시 가능)

**③ 결제 정보**
- 모든 결제는 Apple App Store를 통해 처리되며, 개발자는 카드 정보 등 결제 수단 정보에 접근할 수 없음
- 구독 관리·환불은 Apple 계정 설정에서 가능

**④ 푸시 알림**
- 주간 복기 알림(로컬 알림). 사용자가 iOS 설정에서 언제든 끌 수 있음

**⑤ 아동 개인정보**
- 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않음

**⑥ 이용자 권리와 데이터 삭제**
- 앱 설정 > "모든 데이터 초기화"로 즉시 전체 삭제 가능 (개발자 서버에 사본 없음)
- iCloud 데이터는 사용자 Apple 계정 관리로 삭제 가능

**⑦ 문의처·시행일**
- 문의: seed@arcseed.kr (실제 주소로 교체)
- 방침 변경 시 앱 업데이트 노트로 고지, 시행일 명기

### App Store 개인정보 라벨 (ASC 입력 시)
- **수집하는 데이터: "데이터가 수집되지 않음(Data Not Collected)"으로 선언 가능한지 검토 필요.**
  튜터 질문은 "사용자 콘텐츠 > 기타 사용자 콘텐츠 / 앱 기능 목적 / 사용자와 연결되지 않음 / 추적 없음"으로 신고하는 것이 안전한 선택 (질문 텍스트가 서버를 거치므로)

## 1-A. AI 튜터 조항 — 붙여넣기용 완성 문안 (2026.7 작성)

> 아래를 기존 개인정보처리방침(arcseed.kr/ko/privacy)에 그대로 추가. "제N조" 번호만 문서 체계에 맞게 조정.

### 제N조 (SEED 앱의 AI 튜터 기능)

① 회사가 제공하는 모의투자 학습 앱 'SEED'의 AI 튜터는 이용자가 입력한 질문에 대한 답변을 생성하기 위해, **이용자가 직접 입력한 질문 텍스트를 외부 서버로 전송**합니다. 이는 SEED 앱에서 기기 외부로 정보가 전송되는 유일한 기능입니다.

② 전송되는 정보와 처리 경로는 다음과 같습니다.

| 구분 | 내용 |
|---|---|
| 전송 항목 | 질문 텍스트, 기기별 익명 식별자(무작위 생성 값) |
| 전송되지 않는 항목 | 이름·이메일·연락처 등 개인 식별 정보, 매매 기록, 학습 진행 상황, 위치 정보 |
| 처리 경로 | 회사 중계 서버(Cloudflare Workers) → AI 서비스 제공사(Anthropic PBC)의 API |
| 이용 목적 | 질문에 대한 답변 생성, 기기별 일일 사용량 제한 |
| 보유 기간 | 질문 텍스트는 답변 생성 후 회사 서버에 저장하지 않으며, 익명 식별자별 사용 횟수만 최대 24시간 보관 후 자동 삭제됩니다 |

③ 기기별 익명 식별자는 앱이 무작위로 생성한 값으로, 이용자의 신원과 연결되지 않으며 과도한 사용을 제한하는 목적으로만 사용됩니다.

④ AI 서비스 제공사(Anthropic PBC, 미국)로 전송된 질문은 답변 생성 목적으로만 처리되며, Anthropic의 API 데이터 정책에 따라 **AI 모델 학습에 사용되지 않습니다.** 이 전송은 국외 이전에 해당하며, 이전받는 자와 목적은 본 항에 명시된 바와 같습니다.

⑤ AI 튜터는 선택 기능입니다. 이용자가 질문을 입력하지 않는 한 어떠한 정보도 전송되지 않으며, 튜터를 사용하지 않아도 앱의 다른 기능 이용에는 제한이 없습니다.

### 제N+1조 (SEED 앱의 기기 내 AI 처리)

SEED 앱의 AI 코치 기능(매매 복기·해설 코멘트)은 Apple Intelligence를 통해 **이용자의 기기 안에서만 생성**되며, 이 과정에서 어떠한 정보도 기기 외부로 전송되지 않습니다.

> 메모: ②의 보유 기간 "24시간"은 서버 KV 카운터 TTL과 일치시킨 값 — 서버 구현이 다르면 문구 조정.
> 현재 게시된 방침(/ko/privacy)에는 이 조항이 아직 없음 · 약관(/ko/terms)에는 구독·환불 조항(§2-③) 추가 필요.

### 필수 항목

**① 서비스의 성격** ← SEED 정체성의 법적 방어선

- **투자 자문·종목 추천·수익 보장을 제공하지 않음.** 앱 내 성과는 실제 투자 수익과 무관하며, 실제 투자의 판단과 책임은 이용자 본인에게 있음
- 금융투자업 인가를 받은 서비스가 아님

**② 이용 조건**
- 회원가입 없이 이용 가능. 앱 데이터는 기기·개인 iCloud에 귀속

**③ 유료 서비스** (Apple 표준 문구 포함)
- SEED Pro: 자동갱신 구독 (월 ₩3,300 / 연 ₩22,000). 현재 기간 종료 24시간 전까지 해지하지 않으면 자동 갱신되며, Apple 계정 설정에서 언제든 해지 가능

- 튜터 리필: 소모성 구매

**④ 지식재산권**

**⑤ 면책 조항**

- 서비스는 "있는 그대로" 제공되며, 중단 없는 제공을 보증하지 않음
- AI 생성 콘텐츠(튜터 답변·코치 코멘트)는 부정확할 수 있으며 투자 조언이 아님

**⑥ 약관 변경·준거법**
- 약관 변경 시 앱 내 고지. 대한민국 법률을 준거법으로 함
